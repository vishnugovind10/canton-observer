# Contributor Covenant Code of Conduct

This project follows the [Contributor Covenant 2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

Contributors must maintain a harassment-free, inclusive environment; use welcoming language; respect differing viewpoints; accept constructive feedback; and focus on what is best for the community. Harassment, discriminatory conduct, personal attacks, and publication of others' private information are unacceptable.

Project maintainers may remove, edit, or reject contributions and temporarily or permanently ban participants for conduct they deem inappropriate. Report conduct privately through the repository owner's GitHub profile. Maintainers will respect reporter privacy and apply fair, proportionate consequences.
